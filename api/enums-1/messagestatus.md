# MessageStatus

## 枚举类型介绍

消息状态

## V2TIM\_MSG\_STATUS\_SENDING

枚举类型描述：消息发送中

枚举值：1

## V2TIM\_MSG\_STATUS\_SEND\_SUCC

枚举类型描述：消息发送成功

枚举值：2

## V2TIM\_MSG\_STATUS\_SEND\_FAIL

枚举类型描述：消息发送失败

枚举值：3

## V2TIM\_MSG\_STATUS\_HAS\_DELETED

枚举类型描述：消息被删除

枚举值：4

## V2TIM\_MSG\_STATUS\_LOCAL\_REVOKED

枚举类型描述：被撤销的消息

枚举值：6
