# MessagePriority

## 枚举类型介绍

消息优先级

## 枚举值列表

| 枚举值名称                    | 枚举类型描述           | 枚举值 |
| ------------------------ | ---------------- | --- |
| V2TIM\_PRIORITY\_DEFAULT | 默认为普通优先级         | 0   |
| V2TIM\_PRIORITY\_HIGH    | 高优先级，一般用于礼物等重要消息 | 1   |
| V2TIM\_PRIORITY\_NORMAL  | 普通优先级，一般用于普通消息   | 2   |
| V2TIM\_PRIORITY\_LOW     | 低优先级，一般用于点赞消息    | 3   |
