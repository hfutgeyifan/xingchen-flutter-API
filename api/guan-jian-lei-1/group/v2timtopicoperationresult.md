# V2TimTopicOperationResult

## 关键类描述

话题操作结果（添加、删除等）

## 参数

### topicID

* 参数类型 : String
* 参数描述：话题 ID

### errorCode

* 参数类型 : int
* 参数描述：返回码

### errorMessage

* 参数类型 : String
* 参数描述：返回信息

### toJson

* 参数类型 : Map< String, dynamic > toJson()
* 参数描述：返回话题操作结果的JSON格式
