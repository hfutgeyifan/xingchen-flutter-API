### resultCode

* 参数类型 : int
* 参数描述 : 返回码

### resultInfo

* 参数类型 : String
* 参数描述 : 返回信息

### toJson

* 参数类型 : Map< String, dynamic > toJson()
* 参数描述 : 返回好友操作结果的JSON格式
