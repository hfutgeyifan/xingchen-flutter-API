# V2TimMessageChangeInfo

## 关键类描述

## 参数

### code

* 参数类型 : int
* 参数描述 : 修改结果码

### desc

* 参数类型 : String
* 参数描述 : 描述信息

### message

* 参数类型 : [V2TimMessage](v2timmessage.md)
* 参数描述 : 修改后的信息

### toJson

* 参数类型 : Map< String, dynamic > toJson()
* 参数描述 : 返回消息修改信息的JSON格式
