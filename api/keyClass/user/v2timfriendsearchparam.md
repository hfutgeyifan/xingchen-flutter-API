# V2TimFriendSearchParam

## 关键类描述

搜索用户参数

## 参数

### keywordList

* 参数类型 : List< String >
* 参数描述 : 搜索的关键字列表，关键字列表最多支持 5 个

### isSearchUserID

* 参数类型 : bool
* 参数描述 : 设置是否搜索 userID 默认为true

### isSearchNickName

* 参数类型 : bool
* 参数描述 : 是否设置搜索昵称 默认为true

### isSearchRemark

* 参数类型 : bool
* 参数描述 : 是否设置搜索备注 默认为true

### toJson

* 参数类型 : Map< String, dynamic > toJson()
* 参数描述 : 返回搜索用户参数的JSON格式
