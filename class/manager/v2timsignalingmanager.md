# V2TIMSignalingManager

## V2TIMSignalingManager功能介绍

提供了信令操作相关接口

## API目录

| API                     | 描述       |
| ----------------------- | -------- |
| accept                  | 接收方接收邀请  |
| addInvitedSignaling     |          |
| addSignalingListener    | 添加信令监听   |
| cancel                  | 邀请方取消邀请  |
| getSignalingInfo        |          |
| invite                  | 邀请某个人    |
| inviteInGroup           | 邀请群内的某些人 |
| reject                  | 接收方拒绝邀请  |
| removeSignalingListener | 移除信令监听   |
